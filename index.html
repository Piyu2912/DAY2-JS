Explain var, let and const?

There are three types of variables in javascript:

  • var
  • let
  • const

Difference stated:

var let const
It is Global or Functional Scoped It is Block Scoped It is Block Scoped
It can be modified later on It can be modified later on It cannot be modified later on
It can be reassigned and redeclared It cannot be reassigned and redeclared It cannot be reassigned and redeclared

What is Hoisting?

Hoisting is the default mechanism where our variables and functions declarations are moved up to top of their scope before code execution.

Only declarations are hoisted not initializations.

What is Temporal Dead Zone?

It is the time between the hoisting and the initialization of variables. Even let and const are hoisted but they are in different space called Temporal Dead Zone
